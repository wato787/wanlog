# 静的アセット

## LINE ログインボタン用アイコン

LINE の利用規約に準拠するため、ログインボタンには公式アイコンを使用してください。

1. [LINE Login button design guidelines](https://developers.line.biz/en/docs/line-login/login-button/) から「Download the LINE Login button template」をクリックして ZIP を取得
2. 解凍し、Web 用のアイコン画像を `line.png` としてこのフォルダに配置

アイコンを配置しない場合も、ボタンは公式カラー（#06C755）と文言「LINEでログイン」で表示されます。
